# Uruchamianie

Uruchom Konsole w Folderze z punktatorem
wpisz komende python app.py
otworz 2 terminal wpisz Windows - ipconfig | Linux - ip a Uzyzkaj IP
Polocz sie z wyswietlacza pod adres [TWOJE_IP]:500/display
Polocz sie z palenem sadziego [TWOJE_IP]:500/admin
